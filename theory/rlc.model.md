# RLC Model of Package Interconnect

High-speed interconnects in IC packaging can be approximated using an RLC model.

Driver → R → L → C → Load

Where:

R = driver resistance  
L = interconnect inductance  
C = pad capacitance  

Wirebond typically has higher loop inductance compared to flip-chip bumps.

This causes:

• Larger ringing  
• Reduced bandwidth  
• Higher signal distortion
